## 알고리즘 개념

### 배열

#### 정의

메모리 상에 원소를 연속하게 배치한 자료구조

#### 성질

- O(1)에 k번째 원소를 확인/변경 가능
- 추가적으로 소모되는 메모리의 양(=overhead)이 거의 없음
- Cache hit rate가 높음 => 강의 다 듣고 찾아보기
- 메모리 상에 연속한 구간을 잡아야 해서 할당에 제약이 걸림

#### 기능

- 임의의 위치에 있는 원소를 확인/변경 = O(1)

- 원소를 끝에 추가 = O(1)
- 마지막 원소를 제거 = O(1)

- 임의의 위치애 원소를 추가/제거, O(N)
